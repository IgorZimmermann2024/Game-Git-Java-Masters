### Задача 1: Модели телефонов и их характеристики

**Цель:** Создать массив объектов, каждый из которых представляет сотовый телефон с базовыми характеристиками.

**Описание задачи:**
- Определите класс `CellPhone` с характеристиками, такими как `model` (модель), `manufacturer` (производитель), и `price` (цена).
- Создайте массив, состоящий из нескольких объектов `CellPhone`, заполненный информацией о различных моделях телефонов.
- Напишите функцию, которая выводит информацию о каждом телефоне из массива на экран.

### Задача 2: Поиск телефона по модели

**Цель:** Реализовать функцию поиска телефона по его модели среди массива телефонов.

**Описание задачи:**
- Используя класс `CellPhone` и массив телефонов из предыдущей задачи, реализуйте функцию `findPhoneByModel`, которая принимает название модели в качестве аргумента и возвращает объект телефона с этой моделью.
- Если телефон с заданной моделью не найден, функция должна возвращать `null` или соответствующее сообщение об отсутствии такой модели.

### Задача 3: Фильтрация телефонов по ценовому диапазону

**Цель:** Создать функцию, которая фильтрует массив телефонов по заданному ценовому диапазону.

**Описание задачи:**
- На основе класса `CellPhone` и массива телефонов реализуйте функцию `filterPhonesByPriceRange`, которая принимает два аргумента: минимальную и максимальную цену.
- Функция должна возвращать новый массив, содержащий только те телефоны, цена которых находится в указанном диапазоне.

### Задача 4: Сортировка телефонов по цене

**Цель:** Отсортировать массив телефонов по цене, от меньшей к большей.

**Описание задачи:**
- Используйте массив телефонов, созданный в задаче 1.
- Реализуйте функцию `sortPhonesByPrice`, которая сортирует телефоны в массиве по их цене.
- Выведите отсортированный массив на экран, чтобы продемонстрировать результат сортировки.
